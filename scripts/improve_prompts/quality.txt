src/ 전체를 스캔하고, 다음 기준으로 코드 품질을 개선해:

1. 함수 길이 50줄 초과 -- 더 작은 함수로 분리
2. 하드코딩된 매직 넘버 -- 상수로 추출
3. 과도한 중첩 (depth 3 이상) -- early return 패턴으로 평탄화
4. try-except Exception -- 구체적인 예외 타입으로 변경
5. 타임아웃 없는 HTTP 요청 -- timeout=30 추가
6. 불필요한 import 제거

각 수정 후 uv run pytest tests/ -x -q 를 실행해서 기존 테스트가 깨지지 않는지 확인해.
깨지면 즉시 롤백하고 다른 접근 방식을 시도해.
한 번에 하나의 함수만 수정하고, 통과하면 다음으로 넘어가.
